# 🎾 Tennis Rotation

公平な試合数追跡型ローテーション管理ツール。

## アルゴリズム
- 試合終了時に、コートにいた4人（or 2人）の試合数を +1
- 次の試合に入るメンバーは「**試合数が少ない人** → 同数なら**待機が長い人**」で自動選出
- 試合数の差が大きい人ほど早く呼ばれる

## 機能
- コート数を自由に増減（最低1）
- シングルス / ダブルス 切替
- 待機リストに「次のゲームで優先」バッジ
- 右上の **均等バッジ**（緑=差1以内 / 黄=差2 / 赤=差3+）タップで全員の試合数を一覧表示
- LocalStorage 自動保存

## ローカル起動
```bash
npm install
npm run dev
```

## ビルド / Render デプロイ
リポジトリを Render に接続すれば、`render.yaml` の設定で自動的に Static Site としてビルド・公開されます。

```bash
npm run build  # dist/ に出力
```
